# Company Knowledge Web

Angular 22 frontend for the Company Knowledge RAG Assistant.

## Requirements

Angular 22 requires Node.js `22.22.3+`, `24.15.0+`, or `26+`.

## Run

```powershell
npm.cmd install
npm.cmd start
```

Default frontend URL:

```text
http://localhost:4200
```

Default API URL:

```text
http://localhost:8080
```
